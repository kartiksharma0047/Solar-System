# 🌌 Solar System Simulation (Three.js)

A visually interactive **3D Solar System simulation** built using Three.js.  
This project demonstrates real-time rendering, planetary motion, and basic orbital mechanics in a browser environment.

🔗 **Live Demo:**  
[Demo Link](https://solar-system-puce-kappa.vercel.app/)

---

## 🚀 Features

- 🌞 Central Sun with dynamic lighting
- 🪐 Planets revolving around the Sun with configurable speeds
- 🔄 Self-rotation and revolution for each planet
- 🌀 Elliptical orbits (parametric motion using `cos/sin`)
- 🪐 Saturn ring implementation
- 🌌 Space background with cube textures
- ⚡ Optimized textures using WebP for better performance

---

## 🛠️ Tech Stack

- **Three.js** – 3D rendering and scene management  
- **JavaScript (ES6+)** – Core logic and animation  
- **Parcel** – Bundler for development and production builds  
- **WebGL** – Underlying rendering engine (via Three.js)  
- **Vercel** – Deployment and hosting
